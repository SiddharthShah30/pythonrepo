## What's Inside

### 1. `1_variables.py` — Variables & Data Types

An introduction to Python's core data types:

* **Strings** — text wrapped in quotes
* **Integers** — whole numbers
* **Floats** — decimal numbers
* **Booleans** — `True` / `False` values

---

### 2. `2_typecasting.py` — Type Casting

Converting variables from one type to another using built-in functions like:

* `str()`
* `int()`
* `float()`
* `bool()`

Also covers using `type()` to inspect variable types.

---

### 3. `3_math_functions_operators.py` — Math & Operators

A reference sheet for Python's arithmetic operators:

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Exponentiation (`**`)
* Modulus (`%`) — finding remainders
* Shorthand assignment operators (`+=`, `-=`, etc.)

---

### 4. `4_input.py` — User Input

Using the `input()` function to interact with users, along with f-strings for formatted output.

Includes exercises:

* **Area of a Rectangle** — calculates area from length and width
* **Shopping Cart** — calculates total cost from item, price, and quantity

---

### 5. `5_if_statements.py` — Conditional Statements

Introduction to `if`, `elif`, and `else` statements.

Covers:

* Age eligibility check
* Driver booking confirmation
* Boolean condition example (`for_sale`)
* Comparison operators (`==`)
* Logical flow control

---

## 1️⃣ `madlibs.py` — Madlibs Game

A command-line Madlibs game!

The user is prompted to enter:

* Name
* Adjective
* Noun
* Animal
* Verb

Then the program creates a funny Python-themed story.

## 2️⃣ `mini_cal.py` — Mini Calculator

A simple command-line calculator that:

* Takes two numbers
* Asks for an operator (`+`, `-`, `*`, `/`)
* Performs the calculation using `if-elif-else` logic
* Handles invalid operator input

Concepts Used:

* Type casting (`float`)
* Conditional statements
* Arithmetic operators

---

## 3️⃣ `temperature_convertor.py` — Temperature Converter

Converts temperature between:

* Celsius ➝ Fahrenheit
* Fahrenheit ➝ Celsius

Features:

* User chooses unit (`C` or `F`)
* Uses formulas with rounding (`round()`)
* Converts and displays formatted output

Concepts Used:

* Mathematical formulas
* Conditionals
* String formatting
* `.upper()` method for input handling

---

## 4️⃣ `weight_convertor.py` — Weight Converter

Converts weight between:

* Kilograms ➝ Pounds
* Pounds ➝ Kilograms

Features:

* User selects unit (`K` or `L`)
* Automatic conversion
* Rounded output
* Input validation

Concepts Used:

* Arithmetic operations
* Conditional logic
* User input handling

## How to Run

Make sure you have Python 3 installed.

Run any file using:

```bash
python filename.py
```

Example:

```bash
python mini_cal.py
```
